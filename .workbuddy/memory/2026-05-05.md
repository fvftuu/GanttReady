## 修复网络图 JS 语法错误 + 总工期偏差

### 拖拽功能不工作 → JS 语法错误
- **根因 1**: `activities.forEach(function(act) { ...` 缺少闭合 `});`，导致 `buildNetworkSvg` 函数解析失败
- **根因 2**: `_netEventOffsets` 未声明直接 `if (!_netEventOffsets)` 读取，在严格/模块模式下抛出 ReferenceError
- **根因 3**: 今日线 SVG 字符串跨行拼接可能含不可见字符
- **修复**:
  - 补全 `});` 闭合 activities.forEach
  - `if (!_netEventOffsets)` → `if (typeof _netEventOffsets === 'undefined')`
  - 今日线和 SVG 返回语句重写为单行拼接

### 总工期偏差
- **根因**: `buildNetworkSvg` 中 `总工期 = max(EF)` 只取最大相对偏移，未计算首尾跨度
- **修复**: 改为 `totalDur = max(ef) - min(es) + 1`（含首尾的绝对工期）
- **附带修复**: `Network.razor` 中 `maxEF` fallback 从 `maxES + t.PlanDuration` 改为 `(t.EarlyStart ?? 0) + t.PlanDuration`

### playwright-cli 验证
- JS 语法通过，无 ReferenceError
- SVG 正常渲染：32个事件节点，总工期=368天
