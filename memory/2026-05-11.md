### 1. mattpocock-skills ⭐⭐⭐⭐⭐
**工程师实战 Skills，质量极高**
- `diagnose` — 排错最高境界：不猜，先建反馈循环 (repro→minimise→hypothesise→instrument→fix)。30秒的非确定循环不如没用，2秒的确定性循环才是超能力。
- `triage` — 用状态机管理issue: needs-triage→needs-info→ready-for-agent→ready-for-human→wontfix
- `to-prd` — 从对话上下文生成PRD，发布到issue tracker
- `to-issues` — PRD→可执行issue分解
- `zoom-out` — 后退一步看大局
- `grill-me` — 让AI审视自身的盲点
- `improve-codebase-architecture` — 提取deep modules
- `tdd` — 测试驱动开发
- `caveman` — 强制用最暴力简单的方式实现
- `write-a-skill` — 教你写skill

### 2. GenericAgent ⭐⭐⭐⭐⭐
**自进化Agent框架，核心仅3K行**
- 核心理念：不预加载技能，让技能在执行中自然结晶
- 每次完成任务自动沉淀为skill，下次同类任务直接调用
- Token高效：<30K上下文 vs 其他agent的200K-1M
- 9个原子工具 + ~100行Agent Loop → 系统级控制
- 自我引导证明：整个repo的git提交全由GenericAgent自动完成

### 3. agentmemory ⭐⭐⭐⭐⭐
**#1 AI Agent持久记忆方案**
- 95.2%检索准确率 (R@5 on LongMemEval-S)
- 92% token减少
- 零外部数据库依赖
- **支持OpenClaw集成**：MCP 43工具 + Plugin (memory slot)
- 安装命令：`npx @agentmemory/agentmemory` + 配置MCP
- 实时查看器：http://localhost:3113

### 4. DeepSeek-Reasonix ⭐⭐⭐⭐
**DeepSeek原生终端AI编程agent**

4. **前缀缓存设计** (DeepSeek-Reasonix) — 这个级别的高性能考虑
