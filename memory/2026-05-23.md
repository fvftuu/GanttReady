# 2026-05-23 分析页微调+AI报告HTML化

## 做了什么

### 1. 分析页 header 高度修复
- 根因：`.page-main` 是 flex column，`.page-header` 默认 `flex-shrink:1`，被嵌套的 `.page-body` 压缩从 50px→36.5px
- 修复：`.page-header` 加 `min-height: 50px` + `flex-shrink: 0`

### 2. 移除打印按钮 & AI按钮样式统一
- 删除分析页 header-right 的打印按钮
- AI设置/AI分析按钮从 `btn-sm` + 内联 margin 改为 `btn-default`/`btn-primary`，与甘特图一致

### 3. AI 进度分析报告改为 HTML 格式
- `ChatPromptBuilder.cs`：prompt 从 markdown 指令改为 HTML 标签指令（h2/h3/table/ul/li/p/span）
- `Analysis.razor`：渲染从 `@_aiReport`（纯文本）改为 `@((MarkupString)_aiReport)`（HTML）
- 新增 `.ai-report` CSS 基类 + 子元素样式（h2蓝竖线/表格圆角/斑马纹/悬停高亮）

### 4. 修复聊天面板 `tool_call_id` 错误
- `AIChatPanel.razor`：`RejectPendingToolCall` 过滤掉了 assistant tool_calls 消息，导致后续 API 调用报 `missing field tool_call_id`。删除该过滤行，保留完整消息序列。
