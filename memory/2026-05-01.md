## 修复导入Bug

### 问题
甘特图"导入"功能点击后报错：`导入出错: An invalid request URI was provided. Either the request URI must be an absolute URI or BaseAddress must be set.`

### 根因
`Program.cs` 中 `builder.Services.AddHttpClient()` 注册了 `HttpClient` 但未设置 `BaseAddress`，而 `Gantt.razor` 和 `Index.razor` 使用 `HttpClient.PostAsync("/api/excel/import/{projectId}", content)` 传的是**相对URI**，导致运行时找不到基地址。

### 修复
在两个文件中注入 `NavigationManager`，使用其 `BaseUri` 构造绝对URI：
- **Pages/Project/Gantt.razor** - 添加 `@inject NavigationManager NavManager`，PostAsync 调用改用 `new Uri(new Uri(NavManager.BaseUri), $"/api/excel/import/{currentProjectId}")`
- **Pages/Index.razor** - 同样修复（已注入 `Navigation`，直接使用 `Navigation.BaseUri`）

### 编译
`dotnet build --no-restore` 通过，0错误0警告。

## 修复删除项目功能

### 问题
首页"删除项目"无反应，且导致后续所有操作失效。

### 根因
`DeleteProjectAsync` 只删除了 Project 实体，但 TaskRelation 的外键使用了 `Restrict` 而非 `Cascade`，导致 SQLite 外键约束冲突抛出异常。异常未处理，传播到 Blazor SignalR 电路导致整个页面崩溃。

### 修复
- **ProjectService.DeleteProjectAsync**: 按依赖顺序逐级删除：TaskRelations → ResourceAssignments → Resources → ColumnDefinitions → Tasks → Project
- **Index.razor DeleteSelectedProject**: 添加 try-catch，异常通过 alert 提示用户而非崩溃

## 修复列开关下拉菜单不显示文字、功能无效

### 问题
甘特图"列设置"下拉面板中：
1. 列名文本不显示
2. 勾选/取消复选框无效

### 根因
1. **复选框绑定模式错误**: 使用 `checked="@GetColumnTempVisible(col.Key)"` + `@onchange="() => ToggleColumnTemp(col.Key)"` 模式。在 Blazor 中，浏览器点击后先切换 DOM 状态 → 触发 onchange → C# 更新 Dictionary → Blazor 重渲染 → checked 被重新计算，产生"双次切换"导致复选框始终回到原始状态。
2. **Dictionary 不可响应**: `Dictionary<string, bool>` 的变更不会触发 Blazor UI 刷新，因为 Dictionary 不实现 INotifyPropertyChanged。

### 修复
- **Gantt.razor**: 
  - 新增 `ColumnInfo` 类（Key/Label/Visible），用 `List<ColumnInfo>` 替代 `Dictionary<string, string>`
  - `columnDefs` 从字典改为对象列表，`columnTemp` 为临时编辑副本
  - 复选框改用 `checked="@col.Visible"` + `@onchange="(ChangeEventArgs e) => OnColumnCheckChange(idx, (bool)(e.Value ?? false))"` 模式
  - 新增 `OnColumnCheckChange(int, bool)` 方法直接设置 `columnTemp[index].Visible`
  - `ToggleColumnPanel` 打开时深拷贝到 `columnTemp`，`ApplyColumnVisibility` 写回 `columnDefs`
- **site.css**: 增强 `.column-panel` 和 `.column-check` 样式，添加显式 `color: #333`、accent-color、hover 效果
