🔧 技术栈总览

  • 后端框架：.NET 8 Blazor Server
  • 数据库：SQLite (EF Core)
  • 图表库：Chart.js 4.4.4 (CDN)
  • JS 互操作：Blazor JSInvokable ↔ netplan.js
  • 分析引擎：CPM调度引擎 (ScheduleEngine)
