## 六、关键算法

**关键路径法（CPM）：**
1. 正向计算：起始节点 → 各节点最早时间（ES, EF）
2. 逆向计算：结束节点 → 各节点最晚时间（LS, LF）
3. 时差计算：总时差 = LS - ES = LF - EF
4. 关键路径：总时差 = 0 的任务组成的最长路径
5. 自由时差：不影响后续任务最早开始的前提下可用时差

**挣值管理（EVM）：**
- PV（计划价值）= Σ(任务预算 × 计划进度%)
- EV（挣值）= Σ(任务预算 × 实际完成%)
- AC（实际成本）= 实际已发生成本
- SPI = EV / PV（进度绩效指数）
- CPI = EV / AC（成本绩效指数）
- SV = EV - PV（进度偏差）
- CV = EV - AC（成本偏差）
- CSI = SPI × CPI（综合绩效指数）
